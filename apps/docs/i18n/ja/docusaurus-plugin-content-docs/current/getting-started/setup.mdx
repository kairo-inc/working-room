# 初期設定

インストールが完了してブラウザで WorkingRoom を開いたら、以下の手順に従って最初のアカウントの初期設定を行ってください。

## 1. サインアップページを開く

最初のサインイン画面から、サインアップページに移動して最初のアカウントを作成します。

<Screenshot alt="WorkingRoom サインアップページ" light="/img/screenshots/signup_light.png" dark="/img/screenshots/signup_dark.png" />

## 2. アカウントを作成する

サインアップフォームに必要事項を入力して送信します。

登録が完了すると、WorkingRoom は作成したアカウントの初期パスワードをダウンロードできる画面を表示します。
**パスワードをダウンロード** を選択してファイルをローカルに保存してください。

<Screenshot alt="WorkingRoom サインインページ" light="/img/screenshots/signin_light.png" dark="/img/screenshots/signin_dark.png" />

ダウンロードしたパスワードは安全な場所に保管してください。最初のサインインにはこの初期パスワードが必要です。

## 3. 初期パスワードでサインインする

サインインページに戻り、登録したアカウント ID とダウンロードした初期パスワードを入力します。

<Screenshot
  alt="WorkingRoom 初期パスワードダウンロード"
  light="/img/screenshots/initpassword_light.png"
  dark="/img/screenshots/initpassword_dark.png"
/>

最初のサインイン後、パスワードをリセットして、ダウンロードした初期パスワードを自分だけが知る新しいパスワードに変更してください。

:::note

デフォルトでは、1 つの WorkingRoom サーバーで作成できるテナントは 1 つのみです。

初期設定が完了した後にそのテナントにユーザーを追加したい場合は、[ユーザーを招待する](/docs/user-guide/user-management) のドキュメントを参照してください。

:::
