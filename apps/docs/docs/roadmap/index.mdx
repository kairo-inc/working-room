# Roadmap

This roadmap outlines the current development direction of WorkingRoom and the areas we plan to expand next. Items are subject to change as priorities evolve.

## Current Focus

The following features are actively in progress:

| Feature                   | Category        | Description                                                                             |
| ------------------------- | --------------- | --------------------------------------------------------------------------------------- |
| Access role management UI | Web UI          | Interface to create and manage roles, policies, and user assignments for access control |
| Deleted file recovery     | File Management | Restore accidentally deleted files from the trash                                       |

## Planned

The following items are planned for upcoming releases, grouped by area.

### Web UI

- **Access role management UI** — Interface to create and manage roles, policies, and user assignments for access control
- **Clipboard file upload** — Paste images or files directly from the clipboard into the workspace
- **In-browser text editor** — Edit markdown and plain text files directly in WorkingRoom

### AI & Agent Experience

- **Custom AI agent configuration** — UI to register and configure your own AI agents
- **Gemini support** — Integration with Google Gemini models alongside existing providers

### File System

- **Deleted file recovery** — Restore accidentally deleted files from the trash
- **Extended file format support** — Preview and work with additional file types (e.g. spreadsheets, presentations)
- **Folder upload** — Drag and drop entire folder structures into the file tree
- **File search** — Full-text and filename search across the file system

### Integrations

- **AWS S3 storage backend** — Store files in your own S3 bucket instead of the default storage

## Future Ideas

Longer-term ideas that are not yet on the immediate roadmap:

- **Multi-tenant deployment** — Run WorkingRoom as a shared platform serving multiple independent organizations
- **Google Auth** — Sign in with Google accounts in addition to email/password
